
# Restaurante Admin

En este es el frontend del proyecto de un sistema administracion de restaurante.

* Administración de Mesas
* toma de pedidos en tiempo real
* Pantalla en para ver los pedidos que llegan
* Caja y Facturación
* Categorias, Productos etc..


# Tecnologías

* Create React App
* Tailwind CSS
* Socket.io
* React Router
* Redux
